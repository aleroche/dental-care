# Restorative Dentistry Page - Technical Plan

## Overview
Create a new page in Next.js App Router at:

app/dental-services/restorative-dentistry/page.tsx

The page presents information about restorative dentistry services using a structured layout with reusable components.

---

## Architecture

### Page Type
- Server Component (default)
- No heavy interactivity except potential video embed

---

## Layout Structure

### 1. Header Section
- Title: "Restorative Dentistry"
- Subtitle: "Dental Services"
- Centered layout
- Reusable component: `PageHeader`

---

### 2. Intro Section (Video + Text)
- Two-column responsive layout:
  - Left: YouTube embed (iframe)
  - Right: Text content
- Mobile: stacked (video on top)

Reusable component:
- `VideoTextSection`

---

### 3. Services Section
- Title: "Most recurrent procedures and services"
- List of services with:
  - Title
  - Description

Reusable component:
- `ServicesList`

Data-driven approach:
- Array of objects

---

### 4. Gallery Section
- 4 images grid
- Responsive:
  - Desktop: 4 columns
  - Tablet: 2 columns
  - Mobile: 1 column

Reusable component:
- `ImageGrid`

---

### 5. CTA / Closing Text
- Centered paragraph text
- Emphasis styling

Reusable component:
- `SectionText`

---

## Styling

- TailwindCSS
- Spacing system:
  - Sections: `py-16 md:py-24`
- Container:
  - `max-w-7xl mx-auto px-4`

---
